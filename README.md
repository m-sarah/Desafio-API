# ETL – Santander Dev Week (Python)

## Descrição
Projeto desenvolvido para demonstrar o fluxo ETL (Extração, Transformação e Carregamento) utilizando Python, conforme proposto no desafio da Santander Dev Week.

## Fluxo ETL
- Extração: leitura de dados a partir de um arquivo CSV
- Transformação: geração de mensagens personalizadas
- Carregamento: salvamento das mensagens em um novo arquivo CSV

## Tecnologias
- Python 3
- Biblioteca csv

## Como executar
Execute o arquivo etl.py para gerar o arquivo mensagens.csv.
